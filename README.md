# 🚀 MeetingPilot

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://meetingpilot.streamlit.app/)

AI-powered meeting assistant that transcribes audio, generates concise summaries, extracts topics, analyzes sentiment, and lets you ask questions about your meeting.

## ✨ Features

- 🎙️ Audio Transcription
- 📝 AI Meeting Summary
- 🏷️ Topic Extraction
- 😊 Sentiment Analysis
- 💬 Ask Questions about the transcript
- 🔑 OpenRouter API Integration

## 📸 Workflow

```
Upload Audio
      ↓
Transcription
      ↓
Summary
      ↓
Topics + Sentiment
      ↓
Ask Questions
```

## 🛠️ Tech Stack

- Streamlit
- Whisper
- OpenRouter
- LangChain

## 🚀 Run Locally

```bash
git clone <repo-url>
cd MeetingPilot
pip install -r requirements.txt
streamlit run app.py
```
